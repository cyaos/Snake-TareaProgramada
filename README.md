## Instrucciones
```
1. Debe tener instalado la IDE MARS antes de poder ejecutar el codigo
2. Luego, desde la IDE, se debe abrir el archivo ---.asm
3. Antes de correr el programa se requiere abrir Keyboard and Display MMIO y el BitmapDisplay. Estos se encuentran en la seccion de tools desde la IDE
4. Asegurarse de presionar el boton "Connect to MIPS"
5. Listo! Compilar y correr
```

## Ajustes
```
Ajustes para el Bitmap Display:
- Unit Width: 8
- Unit Height: 8
- Display Width: 256	
- Display Height: 256	
- Base Address for Display: 0x10008000 ($gp)

En KeyboardAndDisplay, se deben usar las letras WASD en la sección de abajo para poder jugar
```
